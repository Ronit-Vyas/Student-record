# 🎓 DDU Alumni Management System

A full-stack web application developed to manage alumni information for **Dharmsinh Desai University (DDU)**. The system allows alumni to register their professional details, stores the information in a MySQL database, and provides complete CRUD (Create, Read, Update, Delete) functionality for managing records.
This project was developed to gain hands-on experience with frontend development, PHP backend programming, database connectivity, and CRUD operations.

---

# ✨ Features

* 📝 Alumni Registration Form
* 💼 Store professional details in a MySQL database
* 📋 View all registered alumni
* ✏️ Update alumni information
* 🗑️ Delete alumni records
* 🔍 Search and browse alumni details
* 📱 Responsive user interface
* 🎨 Bootstrap-based design
* 🖼️ Alumni profile image support

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/Ronit-Vyas/DDU-Alumni-Management-System.git
```

## 2. Download XAMPP

Download XAMPP in your PC if not available

---

## 3. Move the Project

Copy the project folder into your XAMPP `htdocs` directory.

Example:

```text
C:\xampp\htdocs\DDU-Alumni-Management-System
```

---

## 4. Start XAMPP

Start:

* Apache
* MySQL

---

##54. Import Database

* Open phpMyAdmin
* Create a new database
* Import the provided SQL file (`alumni.sql`)

---

## 6. Configure Database

Update your database credentials in `config.php`.

Example:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "alumni";
```

---

## 7. Run the Project

Open your browser and visit:

```text
http://localhost/DDU-Alumni-Management-System/
```
